# Mountain Rescue 

## 1. Files in This Repository

app/Mountain Rescue.app — the Power Apps application

README.md — setup instructions

## 2. Prerequisites

Access to Power Apps

Access to SharePoint Online

Permission to app/data 


## 3. Step 1: Import the App

Go to Power Apps

Select Apps

Choose Import canvas app

Upload Mountain Rescue.app



Remove or fix any broken data source references

You should expect errors at this stage because the app must be connected to your own SharePoint list.
