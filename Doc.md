# Mountain Rescue - Quick Start Guide

## What This App Does

Mountain Rescue helps rescue teams manage emergency incidents in mountain areas by:

Logging rescue incidents  
Recording the victim location and emergency details  
Checking available volunteers from the list  
Assigning the best volunteers to an incident based on their skill  
Notifying volunteers about their rescue task that been assigned by rescue coordinator to them  
Closing the incident after the rescue is completed  

## How to Log an Incident (Step-by-Step)

Open the app  
Click "Log new incident"  
fill details below  
location  
incident type  
severity level  
number of people involved if needed  
Click “Submit Incident”  
You should now see a new incident ID created  

## How to Assign Volunteers (Step-by-Step)

Open the app  
click "View incident"  
Select an incident  
edit the assigned volunteer if needed or assign  
Click “Update”  
The incident status will update, and volunteers will be notified  

## Common Confusion

### “I cant assign a volunteer"

Make sure the volunteer is marked as available and is not already deployed

### “The incident is missing information”

Make sure the location, incident type, severity level, and timestamp are entered correctly

### “The volunteer did not get a notification”

Notifications are only sent after the assignment is confirmed successfully

### “I can’t edit a closed incident”

Click view incidents then select the one you want and edit then update

## Why It Works This Way

Mountain Rescue uses complete incident details so rescue teams can respond quickly and safely.

Volunteers are assigned based on availability, skills, and location so the most suitable team is sent.

Closed incidents cannot be changed easily because the system needs to keep accurate rescue records.

## System Overview (High-Level)

Users interact through a simple UI.

The app the following data listed below:

Incident ID  
Location  
Incident type  
Severity level  
Timestamp  
Volunteer availability  
Volunteer skills  
Assignment status  
Debrief summary  

Rescue coordinators use this information to assign volunteers and manage rescue progress.

## Key Takeaways

Dispatchers log rescue incidents  
Coordinators assign volunteers  
Volunteers must update their availability status  
Only available volunteers can be assigned to incident.  
Incidents include location, type, severity, and time  
The system helps rescue teams respond faster and more accurately
